Downloads

Every tool is available in multiple formats. The HTML file is the simplest option — open it in any browser and it works immediately, no installation needed. It is also the easiest to inspect, modify, and share.

The desktop apps for Windows and Mac are packaged versions of the same tool that run in their own window outside the browser. They are slightly more app-like in feel but require you to extract a zip file before running. Both formats work completely offline and collect no data.
