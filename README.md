# Simple Education Apps

The tools here work without a network connection, persistent data, or external dependcies of any kind. They are single files that can be downloaded and used locally or right in the browser from the site here. No data is ever sent anywhere and they are very privacy-centric with no ads or logins.
## Access
The landing page for all apps is here:
https://mlittrell-ttu.github.io/mlittrell.github.io/

Individual web apps can be accessed by appending the file name to the following:
https://mlittrell-ttu.github.io/mlittrell.github.io/flashcards.html
